# Библиотека, считающая площадь фигур

## Доступные фигуры:

 1. Круг (задается одним единственным параметром - радиусом)
 2. Треугольник (задается тремя сторонами). Так же можно узнать тип треугольника (остроугольный, прямоугольный или тупоугольный)
 3. Прямоугольник (задается 2мя сторонами). Есть возможность узнать, квадрат ли он
